
# Issues / User Stories

## Doctor
- As a **Doctor**, I can view today's queue so that I consult efficiently.
- As a **Doctor**, I can create a prescription so that a patient gets medicines.

## Patient
- As a **Patient**, I can book an appointment and see status to reduce waiting time.
- As a **Patient**, I can search a doctor by name/specialization.

## Admin
- As an **Admin**, I can add doctors and set roles to control access.
- As an **Admin**, I can see daily appointments and revenue report.
